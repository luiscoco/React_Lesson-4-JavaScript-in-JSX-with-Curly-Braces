# JSX Curly Demo (React 19 + Vite)

A tiny app to practice **JavaScript in JSX with curly braces**:
- Variables and expressions inside JSX text
- Dynamic attributes (src, alt, style)
- Inline styles with **double curlies** (`style={{ ... }}`)
- Objects in JSX (`person.theme`), lists, and function calls

## Run locally

```bash
npm install
npm run dev
```

Then open the displayed local URL in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## Files to explore
- `src/App.jsx` — object literals, expressions, recap
- `src/components/PersonCard.jsx` — dynamic attributes and styles
- `src/components/TodoList.jsx` — function calls and list rendering
